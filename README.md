 ##Repo 사용 계획
 1. "모두의 딥러닝" 강의를 듣고 이에 대한 내용을 정리하여 올린다
    - 파일의 종류는 .md 또는 PPT 파일이나 .docx 문서 이다.
    - lab, lec 를 분류한다.
    - 하루 1개 씩은 꼭 지키자

2. "밑바닥부터 시작하는 딥러닝" 정독 후 각 단원에 대해 정리 후 업로드 예정
3. 각 예제 파일 업로드
    - 텐서플로우 튜토리얼
    - MNIST
    - 기타 등등..
4. 논문을 정리한다.
    - 논문 단어 정리 파일
    - 논문 내용 정리 파일
    - 논문 본문 인용

##사용 규칙
1. 꼭 하루에 한개 이상 업로드
2. 논문은 적어도 일주일 한개
3. 예제 파일은 같은건 3번 이상 반복하기(여기에는 끌어온거 1번, 보면서 한거 1번, 안보고 한거 1번 -> 커밋)
